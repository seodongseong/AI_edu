# Churu - 고양이들의 모임장소

고양이를 키우는 사용자들이 자신의 고양이를 공유할 수 있는 소셜 네트워크 서비스입니다.

## 주요 기능

### 1. 사용자 인증
- 로그인 페이지 (`login.html`)
- 회원가입 페이지 (`register.html`)
- 로컬 스토리지를 활용한 세션 관리

### 2. 메인 피드 (`index.html`)
- 고양이 사진 피드 그리드 뷰
- 좋아요 기능
- 댓글 수 표시
- 사진 업로드 기능

### 3. 피드 페이지 (`feed.html`)
- 전체 피드 리스트 뷰
- 상세한 게시물 정보 표시
- 좋아요 및 댓글 기능

### 4. 프로필 페이지 (`profile.html`)
- 사용자 프로필 정보 표시
- 게시물, 팔로워, 팔로잉 통계
- 프로필 편집 기능
- 사용자 게시물 모음

### 5. 고양이 상담 챗봇
- 고양이 키우기 관련 질문 답변
- 먹이, 건강, 놀이, 행동 등 다양한 주제 지원
- 실시간 채팅 인터페이스

## 기술 스택

- HTML5
- CSS3 (반응형 디자인)
- Vanilla JavaScript
- LocalStorage (데이터 저장)

## 사용 방법

1. **회원가입**
   - `register.html`에서 계정을 생성합니다.
   - 이름, 이메일, 비밀번호를 입력합니다.
   - 반려 고양이 이름을 선택적으로 입력할 수 있습니다.

2. **로그인**
   - `login.html`에서 이메일과 비밀번호로 로그인합니다.
   - 로그인 상태는 브라우저에 저장됩니다.

3. **피드 탐색**
   - 메인 페이지에서 다른 사용자들의 고양이 사진을 볼 수 있습니다.
   - 좋아요 버튼을 눌러 관심을 표현할 수 있습니다.

4. **사진 업로드**
   - 메인 페이지의 "사진 올리기" 버튼을 클릭합니다.
   - 사진을 선택하고 설명을 추가한 후 게시합니다.

5. **프로필 관리**
   - 프로필 페이지에서 자신의 정보를 확인하고 수정할 수 있습니다.
   - 프로필 사진, 이름, 소개, 고양이 이름을 변경할 수 있습니다.

6. **챗봇 사용**
   - 사이드바의 "고양이 상담" 버튼을 클릭합니다.
   - 고양이 키우기에 대한 질문을 입력하면 답변을 받을 수 있습니다.

## 이미지 리소스

- **고양이 이미지**: Unsplash API를 사용하여 실제 고양이 사진을 표시합니다.
- **사용자 아바타**: DiceBear Avataaars API를 사용하여 2D 아바타를 생성합니다.
- **이미지 업로드**: 사용자가 직접 업로드한 이미지는 Data URL로 저장됩니다.

## 주요 기능 상세

### 데이터 관리
- `UserDataManager`: 사용자 데이터 관리
- `PostDataManager`: 게시물 데이터 관리
- `AuthManager`: 인증 관리
- `UIRenderer`: UI 렌더링 관리

### 챗봇 기능
- 고양이 키우기 관련 다양한 주제에 대한 답변 제공
- 자연어 처리 기반 키워드 매칭
- 대화 기록 관리
