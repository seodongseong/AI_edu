# DS Studio

모듈형 디지털 서비스 플랫폼을 개발하는 테크 컴퍼니

## 자동 커밋 및 푸시 설정

### 방법 1: 파일 감시 스크립트 (권장)

파일 변경을 실시간으로 감지하여 자동으로 커밋하고 푸시합니다.

```powershell
# PowerShell에서 실행
.\auto-commit.ps1
```

이 스크립트는 백그라운드에서 실행되며, 파일이 변경될 때마다 자동으로 커밋하고 푸시합니다.
`Ctrl+C`로 종료할 수 있습니다.

### 방법 2: 수동 실행 스크립트

변경사항이 있을 때 수동으로 실행하여 커밋하고 푸시합니다.

```powershell
# PowerShell에서 실행
.\auto-commit-simple.ps1
```

### 방법 3: Git Hook 사용

커밋 후 자동으로 푸시하려면 `.git/hooks/post-commit` 파일이 활성화되어 있습니다.

수동으로 커밋한 후 자동으로 푸시됩니다:

```bash
git commit -m "메시지"
# 자동으로 푸시됨
```

## 주의사항

- 자동 커밋은 모든 변경사항을 포함하므로, 커밋하기 전에 변경 내용을 확인하세요.
- `.git` 폴더와 `auto-commit.ps1` 파일은 자동 커밋에서 제외됩니다.
- 여러 파일이 동시에 변경되면 2초 딜레이 후 한 번에 커밋됩니다.
